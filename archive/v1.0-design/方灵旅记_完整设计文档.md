# 《方灵旅记》中医方剂学习游戏 - 完整设计文档

## 一、项目概述

### 游戏名称
**《方灵旅记》** - 讲述一个方剂师在方灵界独自求索的故事

### 核心理念
- **纯单机体验**：专注个人学习旅程，无社交压力
- **AI原生设计**：大模型深度融入游戏机制，不只是对话
- **开放探索**：没有标准答案，鼓励创造性思维
- **分阶段开发**：模块化设计，可快速上线MVP

### 世界观设定
玩家扮演"方灵师"，在**方灵界**中进行孤独的修行：
- 曾经辉煌的方剂文明已经失落
- 你是一百年来的第一位方灵师
- 通过修复散落的"方剂灵阵"来重建知识体系
- 每修复一个灵阵，就唤醒一位"药灵"成为同伴
- 最终目标是修复传说中的"医圣之境"

**情感基调**：孤独但充实的求道过程，类似《Journey》《光·遇》

## 三、第一阶段详细设计

### 模块1：药灵山谷（收集系统）

#### 核心玩法
在东方幻想风格的山谷中，探索并收集"药灵种子"。

#### 场景设计
**五大区域**：
- 🏔️ **高山区域**：雪莲、川贝、冬虫夏草
- 🌊 **溪边区域**：泽泻、菖蒲、车前子
- 🌳 **林间区域**：人参、灵芝、茯苓
- 🌸 **花田区域**：金银花、菊花、红花
- 🪨 **岩壁区域**：石膏、磁石、龙骨

#### 药灵种子外观
- 半透明水晶质感包裹
- 内部隐约可见药材轮廓
- 发光粒子效果，微微浮动
- 悬停显示药名轮廓（未收集）/完整药名（已收集）

#### 收集机制
- **初始**：新玩家获得20颗随机种子
- **每日**：登录获得3颗新种子
- **探索**：点击"探索"按钮，回答AI生成的小谜题发现新种子
- **成就**：完成方剂试炼获得种子奖励

#### AI生图策略
**三阶段视觉**：
1. **种子状态**：水晶包裹，梦幻神秘
2. **药材图**：真实中药材高清图（AI生成）
3. **药灵人格**：新国风拟人化形象（AI生成）

#### 组合判定
AI判断组合类型：
- 🔵 **经典方剂**（如麻黄汤）→ 双倍奖励
- 🟢 **类方变体**（加减变化）→ 1.5倍奖励
- 🟡 **合理自创**（配伍合理但非经典）→ 基础奖励
- 🔴 **配伍不当**（十八反十九畏）→ AI解释为什么不合理

#### 进入试炼
组合成功后，进入"方剂鉴定仪式"，依次为每味药进行卡牌试炼。

### 模块3：药灵卡牌试炼（核心玩法）

#### 核心概念
药灵的能力被封印在散落的"特性卡牌"中。玩家需要在限定时间内，识别并收集所有符合该药物特点的卡牌。

#### 游戏流程
```
配伍成功 → 进入试炼 → 依次为每味药试炼 → 完成解锁能力
                ↓
         麻黄汤 = 麻黄→桂枝→杏仁→甘草（4连试炼）
```

#### 卡牌类型

**正确卡牌（4-5张）**：
- ✅ "辛温发散" - 性味特点
- ✅ "开腠理，发汗解表" - 核心功效
- ✅ "宣肺平喘" - 次要功效
- ✅ "风寒表实证" - 适用证型
- ✅ "胸闷喘咳" - 对应症状

**干扰卡牌（6-10张）**：
- ❌ "清热泻火" - 错误功效（黄连）
- ❌ "滋阴养血" - 错误功效（当归）
- ❌ "甘温补气" - 错误功效（黄芪）
- ❌ "风热感冒" - 错误证型
- ❌ "自汗盗汗" - 禁忌证

**迷惑卡牌（相似但不同）**：
- ⚠️ "发汗解表" - 正确但不够完整
- ⚠️ "温通经脉" - 这是桂枝的特点，容易混淆

#### 难度设计

| 难度 | 正确 | 干扰 | 时间 | 特点 |
|------|------|------|------|------|
| 简单 | 3张 | 3张 | 45秒 | 颜色区分（正确=金色） |
| 普通 | 4张 | 6张 | 30秒 | 需阅读内容 |
| 困难 | 5张 | 10张 | 20秒 | 含迷惑卡牌 |

## 五、数据结构

### 核心数据模型

```typescript
// 药灵数据
interface MedicineSpirit {
  id: string;
  name: string;           // 药名
  namePinyin: string;     // 拼音
  category: string;       // 分类（解表药、清热药等）
  nature: string;         // 性味

// 药灵试炼配置
interface MedicineTrial {
  medicineId: string;
  difficulty: 'easy' | 'normal' | 'hard';
  correctCards: Card[];      // 正确卡牌
  distractorCards: Card[];   // 干扰卡牌
  confusionCards?: Card[];   // 迷惑卡牌
  timeLimit: number;         // 时间限制
  memoryFragments: MemoryFragment[];
}

// 卡牌
interface Card {
  id: string;
  content: string;        // 卡牌文字
  type: 'correct' | 'distractor' | 'confusion';
  category: 'nature' | 'function' | 'indication' | 'symptom' | 'contraindication';
  explanation: string;    // 选择后的解释
  medicineRef?: string;   // 混淆项指向的药
}

// 记忆碎片
interface MemoryFragment {
  id: string;
  type: 'origin' | 'morphology' | 'case' | 'combination';
  title: string;
  content: string;
  hint: string;
  imagePrompt: string;
}

// 方剂数据
interface Formula {
  id: string;
  name: string;
  namePinyin: string;
  category: string;
  composition: {
    medicineId: string;
    role: 'jun' | 'chen' | 'zuo' | 'shi';
  }[];
  functions: string;
  indications: string;
  unlockStatus: {
    isDiscovered: boolean;
    isActivated: boolean;
    isPowered: boolean;
    isEnlightened: boolean;
  };
  skillTree: Skill[];
}

// 玩家数据
interface Player {
  id: string;
  collectedMedicines: string[];
  unlockedFormulas: string[];
  formulaMastery: FormulaMastery[];
  currency: number;
  achievements: string[];
}

```
为中医初学者生成一个病例：

- 包含患者信息、症状、舌脉

```
为{药名}生成卡牌试炼内容：
- 正确卡牌4张（性味、功效、主治）
- 干扰卡牌6张（其他药功效、禁忌）
- 迷惑卡牌2张（相似药物）
- 记忆碎片3个（起源、形态、配伍）
```

**反馈生成Prompt**
```
玩家选择卡牌：{卡牌内容}

药物：{药名}

生成反馈：
1. 判断正误
2. 解释原因
3. 药灵反应台词
4. 知识点补充（如需要）
```
