# Phase 4 AI导师系统端到端测试设计文档

## 1. 测试目标

验证AI导师系统（青木先生）的对话质量、教学效果和用户体验，确保AI能够：
- 提供结构化、连贯的学习引导
- 有效运用苏格拉底式教学法
- 生成多样化、有针对性的题目
- 保持中医教学的专业性和趣味性

## 2. 评价标准体系

### 2.1 维度A：对话质量评估（40分）

#### A1. 上下文连贯性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 多轮对话保持完美上下文，能引用前文内容 | 9-10 |
| 良好 | 基本保持连贯，偶尔需要重复说明 | 7-8 |
| 及格 | 上下文偶有断裂，但能回到主题 | 5-6 |
| 不及格 | 频繁脱离上下文，每次对话都从零开始 | 0-4 |

**测试方法**：
- 进行3-5轮连续对话
- 检查第N轮是否引用第N-1轮内容
- 验证话题是否逐步深入

#### A2. 教学逻辑性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 逻辑严密，层层递进，符合认知规律 | 9-10 |
| 良好 | 逻辑清晰，偶有跳跃但不影响理解 | 7-8 |
| 及格 | 逻辑基本通顺，部分环节需要改进 | 5-6 |
| 不及格 | 逻辑混乱，前后矛盾 | 0-4 |

**测试方法**：
- 评估从简单到复杂的引导路径
- 检查是否遵循"观察→思考→验证"的认知流程

#### A3. 角色一致性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 完全保持"青木先生"角色，用语风格一致 | 9-10 |
| 良好 | 基本保持角色，偶尔有现代用语混入 | 7-8 |
| 及格 | 角色特征不明显，但专业度尚可 | 5-6 |
| 不及格 | 频繁脱离角色，像普通AI而非导师 | 0-4 |

**角色特征检查点**：
- 使用"徒儿"、"师弟"等称呼
- 引用《伤寒论》《本草纲目》等经典
- 使用"为师"、"老朽"等自称
- 语气温和但专业

#### A4. 表达生动性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 故事化场景，形象生动，引人入胜 | 9-10 |
| 良好 | 较为生动，有场景描述 | 7-8 |
| 及格 | 平实但清晰，缺乏场景感 | 5-6 |
| 不及格 | 干巴巴的知识点罗列 | 0-4 |

**评估要素**：
- 是否有故事化引入
- 是否使用比喻、类比
- 是否有场景描述（如"谷中草庐"、"病人"等）

### 2.2 维度B：教学效果评估（35分）

#### B1. 苏格拉底引导（15分）

**核心测试场景**：

**场景1：首次答错**
```
用户状态：回答错误
期望AI行为：
1. 识别错误类型（概念混淆/记忆错误/理解偏差）
2. 提出启发性问题，引导学生思考
3. 不提供直接答案
```

**场景2：继续困惑**
```
用户状态：引导后仍不理解
期望AI行为：
1. 提供更多提示（如"想想《伤寒论》怎么说"）
2. 换一个角度提问
3. 仍不提供直接答案
```

**场景3：明确要求答案**
```
用户状态："请直接告诉我答案"
期望AI行为：
1. 给出正确答案
2. 附带详细讲解（包括出处、原理）
3. 确认学生理解
```

| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 3轮引导策略完美执行，最终讲解透彻 | 13-15 |
| 良好 | 基本执行引导策略，讲解较清晰 | 10-12 |
| 及格 | 有引导意识，但策略不够系统 | 7-9 |
| 不及格 | 直接给答案或引导无效 | 0-6 |

**关键评价点**：
- 是否识别错误原因（如"四气"和"五味"混淆）
- 引导问题是否有针对性
- 3轮内是否逐步降低难度
- 最终答案是否附带出处和原理

#### B2. 知识点准确性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 100%准确，引用出处正确 | 9-10 |
| 良好 | 基本准确，偶有小的表述问题 | 7-8 |
| 及格 | 有少量错误但不影响理解 | 5-6 |
| 不及格 | 知识点错误较多 | 0-4 |

**验证内容**：
- 药材性味归经是否准确
- 方剂君臣佐使是否正确
- 经典引用是否真实存在
- 病案辨证是否符合中医理论

#### B3. 学习适应性（10分）
| 评分等级 | 标准描述 | 分值 |
|---------|---------|------|
| 优秀 | 完美适配学生水平，既不过难也不过易 | 9-10 |
| 良好 | 基本适配，偶有偏差 | 7-8 |
| 及格 | 难度波动较大 | 5-6 |
| 不及格 | 明显与学生水平不符 | 0-4 |

**评估维度**：
- 问题难度是否符合章节进度
- 是否基于已学内容出题
- 是否考虑学生已收集的药材

### 2.3 维度C：题目生成质量（15分）

#### C1. 题目多样性（5分）

4. 跨章对比题（如"解表药和清热药如何区分？"）

| 不及格 | 单一类型，缺乏变化 | 0-2 |

| 不及格 | 干扰项明显错误或无干扰性 | 0-2 |

## 4. 测试场景设计

**测试流程**：
1. 进入第一章


**测试流程**：

## 5. AI模拟学生设计

### 5.1 学生角色设定

**学渣型**：
- 经常答错基础知识
- 需要多次引导
- 会主动要求答案

**普通型**：
- 有时答对有时答错
- 能跟随引导思考
- 偶尔需要提示

**学霸型**：
- 大多能答对
- 喜欢深入提问
- 会追问原理

### 5.2 对话策略

**模拟学生Prompt模板**：
```
你是一位学习中医的学生，当前状态：
- 已学章节：第{X}章
- 已收集药材：{列表}
- 性格：{学渣/普通/学霸}

与青木先生对话时：
- 学渣型：经常答错，会要求直接给答案
- 普通型：有时对有时错，能跟随引导
- 学霸型：多能答对，喜欢追问原理

请根据老师的问题给出符合你水平的回答。
```

## 7. 反馈优化循环机制

### 7.1 问题分类

**P0级（严重）**：
- 知识点错误
- 完全脱离角色
- 逻辑混乱无法继续

**P1级（重要）**：
- 引导策略不当
- 题目重复率高
- 上下文断裂

**P2级（优化）**：
- 表达不够生动
- 选项设计可优化
- 响应速度待提升

### 7.2 优化策略

**Prompt优化方向**：
1. 强化角色定义（System Prompt）
2. 明确引导策略（Few-shot示例）
3. 添加质量检查（Self-correction）

**参数调整**：
- Temperature：控制创造性（建议0.7-0.8）
- Max Tokens：控制回复长度（建议150-250）
- Top P：控制多样性（建议0.9-0.95）

### 7.3 循环流程

```
测试 → 评分 → 问题分类 → Prompt优化 → 重新测试
  ↑                                              |
  └──────────────────────────────────────────────┘
```

**终止条件**：
- 连续2轮达到S级 → 终止
- 达到5轮未达B级 → 终止并报告问题
- 累计10轮 → 终止
