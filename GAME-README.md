# 药灵山谷 - 完整可玩游戏

🎮 **药灵山谷**是《方灵旅记》的独立体验版本，一个东方幻想风格的中医方剂学习收集游戏。

## ✨ 游戏特色

### 🏔️ 五大探索区域
- **高山区域**：🏔️ 雪莲、川贝、冬虫夏草 - 雪花飘落的冰晶世界
- **林间区域**：🌳 人参、灵芝、茯苓 - 阳光透过树叶的神秘森林
- **花田区域**：🌸 金银花、菊花、红花 - 蝴蝶飞舞的温暖花海
- **溪边区域**：🌊 泽泻、菖蒲、车前子 - 水雾弥漫的宁静溪流
- **岩壁区域**：🪨 石膏、磁石、龙骨 - 矿石微光的神秘岩壁

### 💎 水晶种子系统
- 50味中药，每味都有独特的种子
- 水晶包裹的视觉设计
- 精美的收集动画（破碎→显现→飞入图鉴）
- 粒子效果（雪花、光斑、花瓣、雾气、矿石发光）

### ❤️ 亲密度系统
- 5个亲密度等级：初识→相识→熟悉→亲密→知己
- 与药灵互动提升亲密度
- 解锁药灵形象、记忆碎片、专属故事

### 🤖 AI问答探索
- 中药知识问答题
- 答对获得随机种子和方灵点数
- 本地题库（无需API也能玩）
- 支持接入GLM API生成无限题目

### 🎵 音效系统
- 悬停音效（水晶音）
- 收集音效（破碎+光芒）
- 成功/失败反馈音效
- 按钮点击音效

### 📊 进度系统
- 药灵图鉴收集
- 每日登录奖励
- 探索次数限制与重置
- 本地存档自动保存

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
./start-game.sh
```

### 方法二：手动启动

```bash
cd src
npm install
npm run dev
```

游戏将在 http://localhost:3000 打开

## 🎮 游戏玩法

1. **探索山谷**：在五大区域中点击水晶种子进行收集
2. **切换区域**：点击顶部导航栏或使用左右箭头键切换区域
3. **AI问答**：点击"探索"按钮回答问题，获得随机种子
4. **查看图鉴**：点击右上角📖按钮查看已收集的药灵
5. **培养亲密度**：与药灵对话，提升亲密度等级
6. **每日登录**：每天登录可获得种子和方灵点数奖励

## 🖼️ 图像系统

游戏包含两种图像模式：

### SVG占位符（默认）
- 精美的新国风风格SVG动画
- 无需外部图像资源即可运行
- 流畅的60fps动画效果

### AI生成图像（可选）
配置环境变量后可自动生成：
- 五大区域背景图
- 药灵种子高清图
- 药灵人格化形象

## 🔧 配置AI服务（可选）

创建 `.env.local` 文件：

```env
# 文本生成（GLM）
VITE_GLM_API_BASE=https://ark.cn-beijing.volces.com/api/v3
VITE_GLM_API_KEY=your_key
VITE_GLM_MODEL_NAME=glm-4

# 图像生成（Seedream）
VITE_SEED_IMAGE_URL=https://ark.cn-beijing.volces.com/api/v3/images/generations
VITE_SEED_IMAGE_KEY=your_key
VITE_SEED_MODEL_NAME=doubao-seedream-4-5-251128

# 图像理解（Qwen-VL）
VITE_QWEN_VL_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
VITE_QWEN_VL_KEY=your_key
VITE_QWEN_VL_MODEL_NAME=qwen-vl-max
```

## 🎯 技术亮点

- **React 18 + TypeScript**：类型安全的现代前端
- **Zustand**：轻量级状态管理，支持持久化
- **Framer Motion**：流畅的动画效果
- **Canvas API**：高性能粒子系统
- **Tailwind CSS**：原子化CSS，快速开发
- **Web Audio API**：程序化生成音效
- **SVG占位符**：无需外部资源即可运行

## 📝 数据内容
