## 音效设计参考

虽然无法直接生成，但可以描述风格：

| 事件 | 风格描述 |
|------|----------|
| 种子悬停 | 清脆的水晶音，高音，短促 |
| 种子收集 | 玻璃破碎声 + 风铃声 |
| 探索成功 | 古琴 + 编钟，上升音阶 |
| 探索失败 | 低沉木鱼，轻微警示 |
| 界面切换 | 纸张翻动声 |
| 按钮点击 | 玉石轻击声 |
