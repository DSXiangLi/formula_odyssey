### 2.2 阶段详情

**阶段1：师导入门**
- AI导师讲解本章学习目标
- 介绍要学习的4味药材
- 预告2个方剂
- 给出学习建议

**阶段2：山谷采药**
- 进入五行山谷地图
- 探索6x6网格地图
- 发现并采集4味药材
- 每味药材一个小游戏

**阶段3：药灵守护**
- 触发打字战斗
- 4波战斗（药名→性味→功效→方剂）
- 巩固刚学的知识
- 确定药材品质

**阶段4：方剂学习**
- AI导师讲解方剂组成
- 君臣佐使可视化
- 功效主治讲解
- 方歌学习（可选）

**阶段5：临床考核**
- 2-3个病案
- 辨证→选方→选君药
- AI现场点评
- 必须全部正确

**阶段6：开放世界解锁**
- 解锁新区域
- 获得技能奖励
- 可选择继续下一章或自由探索

## 3. 解锁机制

```

```

interface UnlockCondition {
  // 完成任一章，解锁下一章
  // 例如：完成第1章，解锁第4章

  // 多章并行
  // 第1-3章可以任意顺序完成
  // 第5-8章需要完成第1-4章中的任意2章

// 解锁规则
const unlockRules = {
  'ch-004': ['ch-001', 'ch-002', 'ch-003'], // 完成前三章任一解锁第四章

  'ch-009': ['ch-005', 'ch-006', 'ch-007', 'ch-008'], // 完成第5-8章中任意2章解锁第九章
