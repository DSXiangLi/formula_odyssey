# 开放世界设计

## 1. 定位与功能

### 1.1 核心定位

**学习完成后的"自由实践场"**

- 应用已学知识
- 自由探索已解锁区域
- 日常任务与奖励
- 深度养成

### 1.2 与章节的区别

| 维度 | 章节 | 开放世界 |
|------|------|----------|
| 目标 | 结构化学习 | 自由实践 |
| 节奏 | 线性推进 | 自由探索 |
| 压力 | 考核压力 | 放松娱乐 |
| 奖励 | 技能解锁 | 资源积累 |
| 时间 | 45-60分钟 | 碎片时间 |

## 2. 解锁机制

### 2.1 区域解锁

```
完成第1章 → 解锁青木林开放世界
完成第5章 → 解锁赤焰峰开放世界
完成第9章 → 解锁黄土丘开放世界
完成第13章 → 解锁白金原开放世界
完成第17章 → 解锁黑水潭开放世界
```

### 2.2 解锁内容

每个区域解锁后包含：
- 🗺️ 大地图探索（10x10）
- 👥 患者救治系统
- 📅 每日事件
- 🎯 特殊挑战
- 💎 隐藏宝藏

## 3. 患者救治系统

### 3.1 患者生成

```typescript
interface Patient {
  id: string;
  name: string;
  avatar: string;
  location: string;           // 所在区域

  // 病情
  symptoms: string[];
  tongue: string;
  pulse: string;
  description: string;

  // 答案（AI动态生成）
  correctAnswer: {
    treatment: string;
    formula: string;
    junMedicine: string;
  };

  // 限制
  requiredMedicines: string[];  // 需要已收集的药材
  requiredFormulas: string[];   // 需要已学会的方剂

  // 奖励
  rewards: {
    diamonds: number;
    reputation: number;
    affinity?: Record<string, number>;
  };

  // 时间
  expireTime: Date;           // 患者离开时间
}
```

### 3.2 救治流程

```
发现患者
    ↓
查看病情（症状、舌脉）
    ↓
辨证（选择证型）
    ↓
选方（选择方剂）
    ↓
选药（选择君药）
    ↓
AI点评
    ↓
获得奖励
```

## 5. 探索系统

### 5.1 大地图

```
比章节地图更大（10x10 = 100地块）
隐藏区域需特殊条件解锁
随机刷新珍稀药材
```

### 5.2 探索内容

- **普通地块**：可能发现普通药材
- **隐藏地块**：需要工具/技能解锁，有珍稀药材
- **事件点**：触发随机事件
- **传送点**：快速移动

### 5.3 珍稀药材

```
开放世界特有：
· 千年人参（第9章后开放世界）
· 天山雪莲（第13章后开放世界）
· 海底珍珠（第17章后开放世界）

用途：
· 制作高级丹药
· 兑换稀有皮肤
· 提升技能等级
```

## 6. 技能养成

### 6.1 技能树

```
【探索类】
望气之眼 → 风水感知 → 寻宝雷达
识药精通 → 药性洞察 → 百草通神

【采集类】
熟练采集 → 精准采集 → 大师采集
工具维护 → 工具强化 → 神器打造

【战斗类】
快速输入 → 准确输入 → 神级输入
连击掌握 → 连击大师 → 连击之神

【通用类】
过目不忘 → 博闻强识 → 学富五车
药灵亲和 → 药灵共鸣 → 药灵附体
```

### 6.2 技能升级

```typescript
interface SkillSystem {
  // 获得技能点
  gainSkillPoint(source: SkillPointSource): void;

  // 升级技能
  upgradeSkill(skillId: string): boolean;

  // 技能效果
  applySkillEffects(): void;
}

type SkillPointSource =
  | 'chapter_complete'    // 完成章节
  | 'battle_victory'      // 战斗胜利
  | 'patient_cured'       // 救治患者
  | 'daily_login'         // 每日登录
  | 'achievement';        // 成就解锁
```

## 7. 经济系统

### 7.1 收入来源

| 来源 | 数量 | 频率 |
|------|------|------|
| 每日登录 | 50💎 | 每日 |
| 救治患者 | 50-300💎 | 每日3-5次 |
| 完成事件 | 100-500💎 | 每日1-2次 |
| 探索发现 | 10-50💎 | 随机 |
| 战斗胜利 | 50-200💎 | 无限 |

### 7.2 支出用途

- 购买道具（工具、消耗品）
- 请教青木先生（提示）
- 复活/重试
- 解锁皮肤
- 购买珍稀药材

### 7.3 声望系统

```
救治患者 → 声望+1
完成事件 → 声望+5
瘟疫救援 → 声望+20

声望等级：
· 无名小卒（0-50）
· 初出茅庐（51-200）
· 小有名气（201-500）
· 一方名医（501-1000）
· 名满天下（1001-2000）
· 一代神医（2000+）
```
