# 山谷采药冒险 - 游戏策略设计

## 1. 核心循环

```
进入山谷 → 探索地块 → 发现药材 → 识别药材 → 采集小游戏 → 获得药材
    ↑                                                             ↓
    └────────────── 继续探索 / 返回医馆 ←─────────────────────────┘
```

## 2. 地图系统

### 2.1 地图规格

```typescript
// 每章一个专属地图
interface MapConfig {
  size: 6;                    // 6x6 网格（36个地块）
  terrainTypes: Terrain[];    // 该章节特色地形
  medicineSpots: number;      // 药材刷新点数量（4个主要+2个隐藏）
  eventPoints: number;        // 事件点数量（3-5个）
  timeLimit: number;          // 探索时间限制（20分钟）
}
```

### 2.2 地块类型

| 图标 | 类型 | 描述 | 发现率 | 特色 |
|------|------|------|--------|------|
| 🌿 | 草地 | 开阔草地，易行走 | 20% | 常见药材 |
| 🌲 | 树林 | 茂密树林，视野受限 | 30% | 木类药材 |
| 💧 | 水域 | 溪流湖泊，需绕行 | 15% | 水生药材 |
| ⛰️ | 山地 | 崎岖山路，消耗体力 | 25% | 矿物/根茎 |
| 🎋 | 竹林 | 青木林特色 | 20% | 竹类药材 |
| 🌸 | 花田 | 花海药田 | 35% | 花类药材 |
| 🏔️ | 雪峰 | 黑水潭特色 | 20% | 珍稀药材 |
| 🔥 | 温泉 | 赤焰峰特色 | 15% | 热性药材 |

### 2.3 探索机制

**移动消耗**：
```
普通地块：1点体力
困难地块（山地/水域）：2点体力
特殊地块（雪峰/温泉）：3点体力

初始体力：20点
体力恢复：返回医馆后完全恢复
```

**视野系统**：
```
未探索地块：灰色迷雾
已探索地块：正常显示
当前位置：高亮闪烁
相邻地块：可见轮廓
```

## 3. 药材发现机制

### 3.1 药材分布

每章固定6个药材点：
- 4个**主要药材**：必出本章学习药材
- 2个**隐藏药材**：随机出已学/未学药材

```typescript
interface MedicineSpawn {
  medicineId: string;         // 药材ID
  position: Position;         // 固定位置（每章固定）
  isHidden: boolean;          // 是否隐藏点
  respawnTime?: number;       // 刷新时间（开放世界模式）
}
```

### 3.2 发现流程

```
进入地块
    ↓
触发发现判定（20-35%基础概率）
    ↓
发现药材！
    ↓
AI描述药材外观（3-4个特征）
    ↓
玩家识别（选择/输入药名）
    ↓
识别成功 → 进入采集
识别失败 → 可花5钻石查看提示
```

**识别界面**：
```
🌿 你在草丛中发现了一株药材...

【描述】
· 茎直立，高约30-40厘米
· 叶片披针形，边缘有细锯齿
· 夏季开淡紫色小花
· 揉搓叶片有浓郁香气

【这是何药？】
[紫苏] [薄荷] [荆芥] [香薷]
或直接输入药名：_______

[花5💎 询问青木先生]
```

## 4. 采集小游戏

### 4.1 植物类 - 挖掘时机

**玩法**：按住按钮控制力度条

```
┌──────────────────────────────────┐
│  ⛏️ 挖掘：人参                    │
│                                   │
│  力度                             │
│  ├─太轻─[░░░░░░░░░░]─挖不断      │
│  ├─刚好─[████████░░]─完美！      │
│  └─太重─[██████████]─损伤药效    │
│       ▲                           │
│    当前力度                       │
│                                   │
│  【按住挖掘】 松开确定            │
└──────────────────────────────────┘
```

**判定区间**：
- 完美区（绿色）：60%-80% 力度 → 极品
- 良好区（黄色）：50%-60% 或 80%-90% → 上品
- 及格区（白色）：40%-50% 或 90%-100% → 中品
- 失败区（红色）：<40% 或 >100% → 下品/失败

### 4.2 矿物类 - 敲击节奏

**玩法**：跟随节奏点击

```
┌──────────────────────────────────┐
│  🔨 敲击：石膏                    │
│                                   │
│  🎵 节奏条 🎵                     │
│  ○  ●  ○  ●  ○                   │
│  ↑     ↑     ↑                    │
│  点击！ 点击！ 点击！              │
│                                   │
│  连击：15  准确率：95%            │
└──────────────────────────────────┘
```

**判定**：
- Perfect：误差<50ms → 2分
- Good：误差<100ms → 1分
- Miss：误差>100ms → 连击中断

**品质计算**：
- 总分>90%：极品
- 总分>80%：上品
- 总分>60%：中品
- 总分<60%：下品

### 4.3 动物类 - 追踪套索

**玩法**：滑动套索套住移动的动物

```
┌──────────────────────────────────┐
│  🦌 追踪：鹿茸                    │
│                                   │
│        🦌 跑动了！                │
│         ↓                         │
│    ○────┘ 套索                    │
│   /                               │
│                                   │
│  【滑动抛出套索】                  │
│  力度：[━━━━━━░░░░]              │
│  角度：[━━━━░░░░░░]              │
└──────────────────────────────────┘
```

**操作**：
- 拖拽控制力度和角度
- 松手抛出套索
- 动物会AI躲避
- 套住后不能用力过猛（保护动物）

## 5. 工具系统

### 5.1 基础工具

| 工具 | 功能 | 初始 | 升级效果 |
|------|------|------|----------|
| ⛏️ 药锄 | 挖掘根茎 | ✓ | 力度条减速 |
| ✂️ 药剪 | 采摘花叶 | ✓ | 扩大完美区 |
| 🔨 药槌 | 敲击矿石 | ✓ | 节奏判定放宽 |
| 🧺 药篓 | 存放药材 | ✓ | 容量+5 |

```

  └─ 完美区扩大10%

  └─ 容量15 → 20 → 30
  └─ 稀有药材额外槽位

  └─ 显示距离最近的3个药材
  └─ 显示隐藏药材大致方位
```

## 6. 时辰与天气

### 6.1 时辰系统

```
┌─────────────────────────────────────────┐
│              十二时辰                    │
├──────────┬──────────┬──────────────────┤
│ 时辰     │ 现代时间  │ 采集加成          │
├──────────┼──────────┼──────────────────┤
│ 卯时     │ 05-07时  │ 花类药材+30%      │
│ 辰时     │ 07-09时  │ 露水未干，品质+1  │
│ 巳时     │ 09-11时  │ 标准              │
│ 午时     │ 11-13时  │ 休息为佳，-20%    │
│ 未时     │ 13-15时  │ 叶类药材+30%      │
│ 申时     │ 15-17时  │ 根茎类+30%        │
│ 酉时     │ 17-19时  │ 必须返回          │
│ 戌时     │ 19-21时  │ 危险，迷路概率+50%│
└──────────┴──────────┴──────────────────┘
```

**日落机制**：
- 酉时（17:00）系统提示：「天色将晚，该返回了」
- 戌时（19:00）还在山谷 → 触发「迷路事件」
- 迷路后果：随机损失1-3个已采集药材

### 6.2 天气系统

| 天气 | 图标 | 效果 | 适合采集 |
|------|------|------|----------|
| 晴朗 | ☀️ | 标准 | 所有 |
| 多云 | ⛅ | 体力消耗-10% | 长时间采集 |
| 小雨 | 🌧️ | 发现率+20%，移动-10% | 菌类、根茎 |
| 雨后 | 🌦️ | 隐藏药材出现 | 灵芝、茯苓 |
| 大雾 | 🌫️ | 视野-50%，隐藏区开放 | 珍稀药材 |
| 大风 | 🌬️ | 采集难度+20% | 坚韧药材 |
| 下雪 | ❄️ | 移动-30%，热性药材+50% | 附子、肉桂 |

## 7. 随机事件

### 7.1 事件触发概率

```
每移动3个地块：30%概率触发事件
已触发事件地块：24小时内不再触发
```

### 7.2 事件列表

| 事件 | 概率 | 效果 |
|------|------|------|
| 🧓 采药老人 | 25% | 获得1个隐藏药材位置 / 技巧提示 |
| 🐍 毒蛇出没 | 15% | 小游戏：躲避/驱赶，失败扣10体力 |
| 🦌 灵鹿指引 | 10% | 跟随灵鹿，发现稀有药材 |
| 🌧️ 天气突变 | 20% | 随机天气变化 |
| 🎒 发现遗落 | 15% | 获得随机道具/工具升级材料 |
| 🏛️ 古迹发现 | 10% | 解锁背景故事，获得方灵石 |
| 🌪️ 山风骤起 | 5% | 被吹到随机位置 |

### 7.3 采药老人对话示例

```
🧓 采药老人：「小伙子，这青木林我采了几十年了。
              看你面生，送你个消息——
              东边竹林深处，有株百年人参今日成熟。
              记住，采参要系红绳，不然它会跑！」

【获得】隐藏药材位置 x1
【获得】采集技巧：人参挖掘小游戏力度-10%
```

## 10. 与战斗系统的衔接

```
采集完成（获得4味药）
    ↓
自动触发「药灵守护战」
    ↓
使用刚采集的药材进行打字战斗
    ↓
战斗胜利 → 药材品质确定
    ↓
进入方剂学习阶段
```

**品质传承**：
- 采集中品以上药材 → 战斗中对应药材攻击力提升
- 极品药材 → 解锁特殊技能「药灵附体」
