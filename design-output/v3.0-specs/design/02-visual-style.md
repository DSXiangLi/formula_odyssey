# 视觉风格指南

## 1. 整体风格

### 1.1 风格定位

**东方幻想 + 中医药文化 + 游戏质感**

```
关键词：
· 古典 - 中医文化底蕴
· 神秘 - 五行玄幻色彩
· 自然 - 山水草木元素
· 精致 - 现代游戏品质
· 和谐 - 阴阳五行平衡
```

### 1.2 视觉层次

```
第一层：背景（五行场景）
  └── 氛围营造，沉浸体验

第二层：UI框架（半透明面板）
  └── 信息承载，不影响背景

第三层：内容（卡片、文字）
  └── 清晰可读，重点突出

第四层：特效（粒子、光效）
  └── 增强反馈，提升质感

第五层：弹层（对话框、提示）
  └── 最高层级，吸引注意
```

## 2. 场景设计

### 2.1 青木林（木行）

**氛围**：生机盎然，竹林幽静

**元素**：
- 翠绿竹林，光影斑驳
- 晨雾缭绕，露珠晶莹
- 溪流潺潺，石径蜿蜒
- 药草点缀，蝴蝶飞舞

**色调**：
- 主色：#2E7D32（深青）
- 辅色：#81C784（嫩绿）
- 背景：#E8F5E9（淡绿）

**特效**：
- 飘落竹叶
- 飞舞花粉
- 阳光光斑
- 雾气流动

**特效**：

## 5. 特效设计

### 5.1 粒子系统

**环境粒子**：
```
青木林：竹叶、花粉、光斑
赤焰峰：火星、蒸汽、余烬
黄土丘：麦穗、尘土、炊烟
白金原：霜花、雪花、反光
黑水潭：冰晶、极光、水泡
```

**UI粒子**：
```
成功：金色星星爆发
失败：红色碎片散落
升级：彩色光柱
获得：物品飞入背包
```

### 5.2 光效设计

**按钮光效**：
- 悬停：柔和光晕
- 点击：闪烁一下
- 禁用：暗淡无光

**选中光效**：
- 边框发光
- 轻微放大
- 阴影加深

**提示光效**：
- 呼吸灯效果
- 引导注意
- 柔和不刺眼

### 5.3 过渡效果

**页面切换**：
```css
/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 滑入 */
@keyframes slideIn {
  from { transform: translateX(100%); }
  to { transform: translateX(0); }
}
```

**弹窗出现**：
```css
/* 缩放出现 */
@keyframes scaleIn {
  from { transform: scale(0.8); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}
```

## 9. 视觉一致性

### 9.1 检查清单

- [ ] 色彩是否符合五行主题

- [ ] 动效是否流畅自然
- [ ] 图片质量是否达标
- [ ] 特效是否不过度
- [ ] 整体风格是否一致

### 9.2 设计评审

**评审要点**：
1. 是否符合中医文化氛围
2. 是否保持游戏品质
3. 是否用户体验友好
4. 是否技术可实现
