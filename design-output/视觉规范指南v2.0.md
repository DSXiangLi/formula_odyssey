
### 5.2 持续时间

| 类型 | 时长 | 用途 |
|------|------|------|
| 即时 | 100ms | 按钮点击、状态切换 |
| 快速 | 200ms | 悬停效果、微交互 |
| 正常 | 300ms | 弹窗开关、页面切换 |
| 慢速 | 500ms | 复杂动画、场景过渡 |
| 持续 | 2000-4000ms | 呼吸动画、粒子效果 |

### 5.3 入场动画

```css
/* 淡入 */

  from { opacity: 0; }
```

## 九、设计原则

1. **新国风美学**: 融合传统水墨与现代简约
2. **五行色彩**: 严格遵循五行色彩体系
3. **专业准确**: 中医药知识必须准确
4. **沉浸式体验**: 营造东方幻想氛围
5. **清晰层级**: 信息架构清晰，易于理解
6. **微交互**: 细腻的动效提升体验
7. **一致性**: 保持全游戏视觉一致性
