# 药灵山谷 - 美术规范指南

## 整体风格

**东方幻想 + 精致游戏美术 (Oriental Fantasy + Polished Game Art)**
- 融合东方美学与高品质游戏视觉效果
- 水晶元素增加幻想感
- 微光粒子营造神秘氛围
- **注意：不是水墨画风格，而是精致的游戏场景原画**

## 场景背景风格规范

### 核心要求
- **风格定位**: Oriental Fantasy / 东方幻想
- **品质标准**: AAA级游戏概念美术
- **视觉效果**: 精致、吸引人、一眼抓住用户
- **元素特征**:
  - 水晶/魔法元素 (crystal formations)
  - 发光植物/粒子 (luminous flora, magical particles)
  - 电影级光影 (cinematic lighting)
  - 景深效果 (depth of field)
  - 空灵氛围 (ethereal glow)

### 五大区域风格关键词

**青木林 (木)**
- 水晶竹林、发光藤蔓、生物荧光花朵
- 晨雾中舞动的魔法萤火虫
- 充满生命力的春之魔法

**赤焰峰 (火)**
- 红色水晶、熔岩魔法玻璃
- 飘浮的余烬、火焰百合
- 凤凰羽毛散落在地

**黄土丘 (土)**
- 金色水晶麦田、发光石圈
- 漂浮的大地微粒
- 巨型金色蘑菇、丰收精灵

**白金原 (金)**
- 银色水晶桦树、漂浮的银叶
- 铂金矿脉发光
- 镜面冰池、水晶石英

**黑水潭 (水)**
- 黑曜石深潭、生物发光水下植物
- 悬浮冰晶、极光般的光芒
- 月光在水面形成银路

## 动画规范

### 过渡时间

| 类型 | 时长 | 用途 |
|------|------|------|
| 微交互 | 150ms | 按钮点击、状态切换 |
| 快速 | 200ms | 悬停效果、小过渡 |
| 标准 | 300ms | 弹窗、面板切换 |
| 慢速 | 500ms | 场景切换、复杂动画 |
| 入场 | 800ms | 页面入场、重大揭示 |

### 缓动函数

```css
:root {
  --ease-out: cubic-bezier(0, 0, 0.2, 1);
  --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  --ease-smooth: cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
```

### 种子收集动画序列

```
总时长: 1.5秒

0ms-200ms: 响应点击
  - 缩放至1.2
  - 亮度最大化

200ms-600ms: 水晶破碎
  - 外壳裂纹扩散
  - 碎片向四周飞散
  - 内部光芒爆发

600ms-1000ms: 精华显现
  - 药材虚影显现
  - 向中心汇聚
  - 药灵形象凝聚

1000ms-1500ms: 飞入图鉴
  - 飞向屏幕右上角
  - 轨迹粒子效果
  - 图鉴按钮闪烁
```
