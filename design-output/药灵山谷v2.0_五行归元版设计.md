### 1.3 五行生克游戏机制

**相生关系（促进）**：
- 木生火：在青木林收集的药材，带到赤焰峰有20%效果加成
- 火生土：在赤焰峰收集的药材，带到黄土丘可直接入药（无需炮制）
- 土生金：在黄土丘合成方剂后，白金原的稀有种子刷新率+50%
- 金生水：在白金原连续收集3个种子，黑水潭必出高品质种子
- 水生木：在黑水潭钓鱼获得的水生药材，在青木林种植可变异为特殊品种

**相克关系（制约）**：
- 木克土：青木林采集后1小时内，黄土丘的种子出现率降低
- 土克水：黄土丘炮制后的药材，在黑水潭区域效果减半
- 水克火：携带黑水潭药材进入赤焰峰，高温会"熄灭"部分药效
- 火克金：赤焰峰"点燃"操作后，白金原的种子会"沉降"更快
- 金克木：白金原采集后，青木林的"生长"机制暂停

## 三、真实药材认知系统

### 3.1 种子视觉设计标准

**水晶种子 = 真实药材照片 + 水晶包裹效果**

#### 三层视觉识别

**第一层：原植物形态**（生长环境）
- 展示药材在自然中的生长状态
- 包含植株整体、叶片、花朵、果实等特征
- 用于"望"诊环节的初步识别

**第二层：药材/饮片形态**（临床使用）
- 展示药房常见的药材形态
- 饮片切片、炮制后的外观
- 用于"闻""切"环节的深入识别

**第三层：显微/细节特征**（专业鉴别）
- 高倍放大后的表面纹理
- 断面特征、粉末显微
- 用于亲密度满级后的"知己"解锁

```
决策场景：

你发现了一颗神秘种子，外观呈圆柱形，黄绿色...

策略A：谨慎型（投入5💎）

- 候选药物：麻黄、桂枝、紫苏、生姜
- 成功率：25%
- 预期回报：30💎（答对）- 5💎（投入）= +25💎
- 风险：答错损失5💎

策略B：平衡型（投入15💎）
- 👁️ 望 + 👃 闻 + ❓ 问 + ✋ 切
- 获得线索：解表药 + 辛香气味 + 东北产 + 辛苦温归肺经
- 候选药物：麻黄
- 成功率：95%
- 预期回报：30💎 - 15💎 = +15💎
- 风险：极低

策略C：赌博型（投入0💎）
- 仅用 👁️ 望
- 候选药物：50味中的任意解表药
- 成功率：<5%
- 预期回报：30💎 - 0💎 = +30💎（但极难成功）
- 风险：极高，但无成本

策略D：全知型（投入50💎）
- 解锁全部信息
- 成功率：100%
- 预期回报：30💎 - 50💎 = -20💎
- 风险：无，但亏损
- 适用：关键任务/稀有药材
```

**中医思维训练：**
- 鼓励玩家用**最少信息准确判断**（模拟中医"但见一症便是"的辨证思维）
- 学习"四诊合参"——单一信息可能误导，需要综合判断
- 理解"同气相求"——气味、颜色、归经的关联性
