# 药灵山谷 v2.0 - 五行归元版设计文档

## 设计理念升级

### 核心设计哲学
**从"收集游戏"升级为"中医专业学习模拟器"**

- 以**五行学说**为骨架：金木水火土对应五脏、五方、五色、五季
- 以**性味归经**为玩法：药图、四气、五味、归经、功效五种探查方式，策略性获取信息
- 以**方剂追缉**为目标：每日明确的方剂收集任务，学以致用
- 以**临床实习**为归宿：将所学知识应用于病案分析

## 一、五行架构系统

### 1.1 五行-五脏-五方映射

| 五行 | 脏腑 | 方位 | 季节 | 代表色 | 区域名称 | 核心特征 |
|------|------|------|------|--------|----------|----------|
| **木** | 肝 | 东 | 春 | 青 | 青木林 | 生发、条达 |
| **火** | 心 | 南 | 夏 | 赤 | 赤焰峰 | 温热、升腾 |
| **土** | 脾 | 中 | 长夏 | 黄 | 黄土丘 | 承载、运化 |
| **金** | 肺 | 西 | 秋 | 白 | 白金原 | 肃降、收敛 |
| **水** | 肾 | 北 | 冬 | 黑 | 黑水潭 | 寒凉、闭藏 |

### 1.2 五行场景设计

#### 青木林（木行·肝）
- **视觉风格**：翠绿竹林，嫩芽初绽，生机勃勃
- **环境元素**：青竹、新芽、藤蔓、春风
- **粒子效果**：飘落的新叶、花粉
- **背景音乐**：竹笛清音，鸟鸣婉转
- **专属机制**：春季生发 - 探索时有几率额外获得线索

#### 赤焰峰（火行·心）
- **视觉风格**：赤红岩壁，温泉蒸汽，阳光炽热
- **环境元素**：红枫、温泉、火山石、夏花
- **粒子效果**：上升的蒸汽、火星
- **背景音乐**：鼓点激昂，热情奔放
- **专属机制**：心主神明 - 首次猜测免费

#### 黄土丘（土行·脾）
- **视觉风格**：金黄麦田，厚重大地，丰收景象
- **环境元素**：麦田、黄土、窑洞、农舍
- **粒子效果**：飘落的麦穗、尘土
- **背景音乐**：埙声厚重，沉稳平和
- **专属机制**：脾主运化 - 线索价格降低20%

#### 白金原（金行·肺）
- **视觉风格**：银白秋霜，金属矿脉，肃杀之气
- **环境元素**：白桦林、矿石、霜花、秋风
- **粒子效果**：飘落的霜花、金属光泽
- **背景音乐**：钟磬清越，肃静庄严
- **专属机制**：肺主肃降 - 可以快速跳过当前种子

#### 黑水潭（水行·肾）
- **视觉风格**：幽蓝深潭，寒冰覆盖，静谧深邃
- **环境元素**：黑石、深潭、冰晶、冬雪
- **粒子效果**：飘落的雪花、冰晶闪烁
- **背景音乐**：古琴低沉，幽静深远
- **专属机制**：肾藏精 - 收集成功后奖励翻倍

### 1.3 药物五行归属

根据药物的性味、归经、功效，将50味药归入五行：

**木行药物（10味）**：归肝经，性多辛温，主疏泄
- 柴胡、香附、郁金、川芎、当归、白芍、地黄、枸杞子、杜仲、续断

**火行药物（10味）**：归心经，性多苦温，主血脉
- 丹参、郁金、川芎、人参、黄芪、桂枝、附子、肉桂、远志、酸枣仁

**土行药物（10味）**：归脾经，性多甘平，主运化
- 人参、黄芪、白术、茯苓、甘草、山药、大枣、山楂、神曲、麦芽

**金行药物（10味）**：归肺经，性多辛凉，主宣降
- 麻黄、桂枝、紫苏、生姜、桑叶、菊花、杏仁、桔梗、贝母、沙参

- 熟地黄、山茱萸、山药、茯苓、牡丹皮、泽泻、附子、肉桂、牛膝、杜仲

## 二、真实药图系统

### 2.1 三层视觉体系

每味药物配备三张专业级图片：

#### 第一层：原植物/矿物/动物图
- **用途**：水晶球中未收集状态的"种子"视觉
- **内容**：药物原生状态（生长中的植物、天然矿物、活体动物）
- **要求**：专业植物志/矿物志摄影风格，准确可辨识

#### 第二层：药材饮片图
- **用途**：收集成功后在图鉴中展示
- **内容**：加工后的中药材（饮片、切片、炮制成品）
- **要求**：《中国药典》标准规格，专业药材摄影

#### 第三层：显微特征图（高级解锁）
- **用途**：亲密度达到"知己"等级后解锁
- **内容**：显微鉴别特征（粉末特征、组织切片）
- **要求**：显微镜摄影，标注特征结构

## 八、数据模型更新

### 8.1 新类型定义

```typescript
// 五行类型
type WuxingType = 'wood' | 'fire' | 'earth' | 'metal' | 'water';

// 四气
type FourQi = '寒' | '热' | '温' | '凉' | '平';

// 五味
type FiveFlavors = '酸' | '苦' | '甘' | '辛' | '咸';

// 升降浮沉
type Movement = '升浮' | '沉降' | '双向' | '平和';

// 药物完整数据
interface MedicineV2 {
  id: string;
  name: string;
  pinyin: string;
  latinName: string;        // 拉丁学名
  category: string;         // 功效分类（解表药等）
  wuxing: WuxingType;       // 五行归属

  // 中药专业属性
  fourQi: FourQi;           // 四气
  fiveFlavors: FiveFlavors[]; // 五味（可多个）
  movement: Movement;       // 升降浮沉
  meridians: string[];      // 归经
  toxicity: string;         // 毒性（有毒/无毒/小毒等）
  functions: string[];      // 功效
  indications: string[];    // 主治
  contraindications: string[]; // 禁忌

  // 图片资源（三层视觉）
  imagePlant: string;       // 原植物/矿物图
  imageHerb: string;        // 饮片图
  imageMicroscope?: string; // 显微图（高级解锁）

  // 故事与亲密度
  stories: string[];
  affinity: number;
  collected: boolean;
}

// 方剂数据
interface Formula {
  id: string;
  name: string;
  pinyin: string;
  category: string;         // 方剂分类（解表剂等）

  composition: {
    medicineId: string;
    amount: string;
    role: 'jun' | 'chen' | 'zuo' | 'shi'; // 君臣佐使
  }[];
  functions: string[];
  indications: string[];
  song?: string;            // 方歌
  variations?: string[];    // 加减变化
  proficiency: number;      // 熟练度 0-5
}

// 方剂追缉令
interface FormulaPursuit {

    currency: number;
    affinityBonus?: number;
    badge?: string;
  };
  expiresAt: string;        // 过期时间
}

// 临床病案
interface ClinicalCase {
  id: string;
  formulaId: string;        // 对应方剂
  patientInfo: string;      // 患者信息
  symptoms: string[];       // 症状
  tongue: string;           // 舌象
  pulse: string;            // 脉象
  correctTreatment: string; // 正确治法
  correctFormula: string;   // 正确方剂
  correctJun: string;       // 正确君药
  explanation: string;      // 解析

// 玩家数据（更新）
interface PlayerV2 {
```
