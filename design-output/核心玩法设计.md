# 药灵山谷 - 独立游戏核心玩法设计

## 游戏概述

**药灵山谷**是《方灵旅记》的独立体验版本，专注于收集和探索玩法。玩家在一个东方幻想风格的山谷中收集药灵种子，解锁药灵图鉴，建立与药灵的亲密度。

### 核心体验
- **探索**：在五大区域自由探索
- **收集**：收集50颗药灵种子
- **养成**：提升药灵亲密度
- **知识**：通过探索问答学习中药知识

---

## 游戏循环

```
进入山谷 → 探索区域 → 发现种子 → 收集种子 → 查看图鉴
    ↑                                           ↓
    └──────────── 每日登录奖励 ←────────────────┘
```

## 五大区域设计

### 1. 高山区域 🏔️
- **环境**：终年积雪，冰川裂缝，寒风呼啸
- **药灵**：雪莲、川贝、冬虫夏草、红景天
- **氛围**：雪花飘落，冰晶发光，空灵音效
- **色彩**：冷白 + 浅蓝 + 冰晶反光

### 2. 林间区域 🌳
- **环境**：古树参天，藤蔓缠绕，苔藓覆盖
- **药灵**：人参、灵芝、茯苓、天麻
- **氛围**：阳光透过树叶，光斑闪烁，鸟鸣虫声
- **色彩**：深绿 + 墨绿 + 金色光斑

### 3. 花田区域 🌸
- **环境**：花海起伏，蝴蝶飞舞，阳光明媚
- **药灵**：金银花、菊花、红花、玫瑰花
- **氛围**：花瓣飘落，蜜蜂嗡嗡，温暖明亮
- **色彩**：粉红 + 明黄 + 橙金

### 4. 溪边区域 🌊
- **环境**：溪流蜿蜒，水雾弥漫，芦苇摇曳
- **药灵**：泽泻、菖蒲、车前子、浮萍
- **氛围**：水面倒影，水汽蒸腾，流水潺潺
- **色彩**：青绿 + 水蓝 + 白色雾气

### 5. 岩壁区域 🪨
- **环境**：陡峭岩壁，洞穴阴影，矿石裸露
- **药灵**：石膏、磁石、龙骨、牡蛎
- **氛围**：矿石微光，回声空灵，神秘庄严
- **色彩**：灰褐 + 赭石 + 矿石荧光

## 药灵种子系统

### 种子视觉

**未收集状态**：
- 半透明水晶球包裹
- 内部隐约可见药材轮廓
- 微微发光，呼吸效果
- 标签显示"???"

**已收集状态**：
- 金色光芒包裹
- 内部清晰可见
- 稳定发光
- 标签显示药名

### 种子分布（每区域10颗）

| 区域 | 种子1 | 种子2 | 种子3 | 种子4-10 |
|------|-------|-------|-------|----------|
| 高山 | 雪莲 | 川贝 | 冬虫夏草 | 随机分布 |
| 林间 | 人参 | 灵芝 | 茯苓 | 随机分布 |
| 花田 | 金银花 | 菊花 | 红花 | 随机分布 |
| 溪边 | 泽泻 | 菖蒲 | 车前子 | 随机分布 |
| 岩壁 | 石膏 | 磁石 | 龙骨 | 随机分布 |

### 收集机制

**初始奖励**：
- 新玩家获得10颗随机种子

**每日奖励**：
- 登录获得3颗随机种子
- 补充3次探索机会

**探索机制**：
- 点击"探索"按钮
- AI生成中药知识问答
- 答对发现新种子
- 答错可重试（消耗时间）

## 亲密度系统

### 等级划分

| 等级 | 分数 | 称号 | 解锁内容 |
|------|------|------|----------|
| Lv.1 | 0-20 | 初识 | 基础信息 |
| Lv.2 | 21-40 | 相识 | 功效详情 |
| Lv.3 | 41-60 | 熟悉 | 药灵形象 |
| Lv.4 | 61-80 | 亲密 | 记忆碎片 |
| Lv.5 | 81-100 | 知己 | 专属故事 |

### 提升方式

- **收集种子**：+10亲密度
- **查看详情**：+5亲密度
- **每日互动**：+5亲密度
- **完成探索**：+15亲密度

---

## 探索问答系统

### 题目类型

1. **识药题**：根据描述猜药名
2. **性味题**：选择正确的性味归经
3. **功效题**：选择正确的功效主治
4. **配伍题**：选择合理的药物组合

### 难度分级

- **简单**：基础知识点，明显选项
- **普通**：需要思考，有相似干扰项
- **困难**：综合运用，迷惑性强

### 奖励机制

- **答对**：获得随机种子 + 方灵点数 + 亲密度
- **答错**：显示解析，可重试

---

## 图鉴系统

### 图鉴界面

- **网格布局**：展示所有药灵
- **分类筛选**：按类别筛选（解表药、清热药等）
- **搜索功能**：按药名、拼音、功效搜索
- **进度显示**：已收集/总数

### 药灵详情页

- **三态切换**：种子 → 药材 → 药灵
- **基本信息**：药名、性味、归经
- **功效主治**：主要功效、适用证型
- **记忆碎片**：起源、形态、配伍故事
- **亲密度展示**：当前等级和进度

---

## 经济系统

### 方灵点数

- **用途**：解锁特殊内容、加速收集
- **获取**：收集种子、完成探索、每日登录

### 探索次数

- **上限**：每日3次
- **重置**：每日0点重置
- **额外**：观看广告或消耗点数获得

## 技术需求

### 需要实现的功能

1. **场景渲染**：Canvas/Three.js渲染五大区域
2. **粒子系统**：雪花、花瓣、光斑、雾气效果
3. **动画系统**：种子收集动画、UI过渡动画
4. **数据存储**：本地存储收集进度和亲密度
5. **AI集成**：GLM API生成问答题目
6. **音效系统**：背景音乐、交互音效
