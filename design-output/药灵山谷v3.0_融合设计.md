### 3.2 单章学习流程

每章约45-60分钟，包含**6个阶段**：

| 阶段 | 名称 | 时长 | 核心玩法 | 说明 |
|------|------|------|----------|------|
| 1 | 师导入门 | 5min | AI对话 | 导师导入学习目标 |
| 2 | 山谷采药 | 15min | 探索+采集小游戏 🎮 | 进入地图采集4味药材 |
| 3 | 药灵守护 | 5min | 打字战斗 ⌨️ | 巩固刚学的药材知识 |
| 4 | 方剂学习 | 10min | AI讲解 | 学习2个方剂组成 |
| 5 | 临床考核 | 10min | 病案辨证 | Boss考核关卡 |
| 6 | 开放世界 | 5min | 解锁新区域 | 自由探索救治 |

#### 阶段1：师导入门（5分钟）

**AI导师场景导入**：
- 青木先生站在对应五行场景中
- 讲述本章学习目标
- 预告本章要学习的药物和方剂

**示例**（第1章）：
```
青木先生：「徒儿，今日我们来到青木林，学习解表之法。
         外感风寒，邪在肌表，须用辛温解表之药。
         本章你将识得麻黄、桂枝等四味解表良药，
         并学习麻黄汤、桂枝汤两个经典方剂。」
```

#### 阶段2：山谷采药（15分钟）⭐ 全新玩法

**核心玩法：山谷采药大冒险**

玩家进入五行山谷地图，探索、发现、采集药材。

**2.2.1 探索系统**

每个五行区域都是一个**可探索地图**：

**青木林（木行）地图示例**：
```
┌─────────────────────────────────┐
│  🌲 古木林   │   🌸 药花园      │
├──────────────┼──────────────────┤
│  🎋 竹林     │   💧 溪涧        │
├──────────────┼──────────────────┤
│  🌿 草地     │   🏔️ 山脚        │
└─────────────────────────────────┘
```

**移动与探索**：
- 点击相邻地块移动
- 每次移动消耗体力
- 进入地块可能发现药材或触发事件

**2.2.2 发现药材**

进入地块后，AI描述环境，玩家需要**识别**药材：

```
🎋 你走进一片竹林，阳光透过竹叶洒下斑驳光影。
   地上长满了各种杂草，其中几株引起了你的注意...

   一株植物茎直立，高约30厘米，叶片披针形，
   边缘有细锯齿，开着淡紫色的小花...

   [这是何药？]

   ┌─────────────────────────────────┐
   │  [薄荷]  [紫苏]  [荆芥]  [香薷]  │
   └─────────────────────────────────┘

   或直接输入药名：________
```

**2.2.3 采集小游戏**

识别成功后，进入**采集小游戏**：

**植物类 - 挖掘时机**：
```
┌─────────────────────────────────┐
│  ⛏️ 挖掘时机小游戏              │
│                                 │
│  力度: [████████░░] 80%        │
│       ▲                       │
│    恰到好处！                  │
│                                 │
│  [按住挖掘]                     │
└─────────────────────────────────┘
```
- 按住按钮力度条增长
- 在绿色区域松手=完美采集
- 力度太小=挖断根须；力度太大=损伤药效

**矿物类 - 敲击节奏**：
- 跟随节奏点击，类似音游
- 连击越高，品质越好

**动物类 - 追踪套索**：
- 动物会移动逃跑
- 滑动套索套住动物
- 不能用力过猛（保护动物）

**采集品质**：

| 品质 | 判定 | 药效加成 |
|------|------|----------|
| 极品 | 小游戏完美 | +50% |
| 上品 | 小游戏良好 | +20% |
| 中品 | 小游戏及格 | 标准 |
| 下品 | 小游戏失败 | -20% |

**2.2.4 采药工具系统**

```
🎒 采药工具包
├─ ⛏️ 药锄    - 挖掘根茎（可升级）
├─ ✂️ 药剪    - 采摘花叶
├─ 🔨 药槌    - 敲击矿石
├─ 🧺 药篓    - 存放药材（容量15）
├─ 🧭 罗盘    - 定位药材方位
├─ 📖 药典    - 识别未知药材
└─ 🍞 干粮    - 恢复体力
```

**2.2.5 时间与天气系统**

**时辰系统**：

| 时辰 | 时间 | 特殊效果 |
|------|------|----------|
| 卯时 | 05:00-07:00 | 采花最佳 |
| 辰时 | 07:00-09:00 | 露水未干 |
| 未时 | 13:00-15:00 | 采叶最佳 |
| 申时 | 15:00-17:00 | 采根最佳 |
| 酉时 | 17:00-19:00 | 必须返回 |
| 戌时 | 19:00-21:00 | 危险！迷路 |

**日落机制**：酉时必须开始返回，戌时还在山中会迷路损失药材。

**天气系统**：

| 天气 | 效果 | 适合采集 |
|------|------|----------|
| ☀️ 晴朗 | 标准 | 所有 |
| 🌧️ 小雨 | 发现率+20% | 菌类、根茎 |
| 🌦️ 雨后 | 特殊药材出现 | 灵芝、茯苓 |
| 🌫️ 大雾 | 视野受限 | 隐藏区域开放 |

**2.2.6 随机事件**

探索中可能触发：

| 事件 | 描述 | 奖励 |
|------|------|------|
| 🧓 采药老人 | 获得情报、技巧传授 | 隐藏药材位置 |
| 🐍 毒蛇出没 | 躲避或驱赶 | 经验/受伤 |

## 四、AI导师系统

### 4.1 角色设定

**名称**：青木先生
**形象**：白须老中医，手持药典，身着对应章节五行色长衫
**性格**：温和耐心，循循善诱，引经据典但不失幽默
**背景**：隐居药灵山谷三百年的医道高人

**视觉呈现**：
- 左侧：导师立绘（5种表情：微笑、思考、点头、惊讶、庆祝）
- 右侧：对话气泡（自然对话形式）
- 底部：玩家回复区域（支持多种交互形式）

### 4.2 对话系统（自然自由）

**核心原则**：不局限于固定选项，支持多种交互形式

#### 交互形式1：选择题（适合初学者）

```
青木先生：「今日来了个病人，恶寒发热，无汗而喘。
         你说说，这是什么证型？」

┌─────────────────────────────────┐
│  A. 风寒表虚证 - 汗出恶风       │
│  B. 风寒表实证 - 无汗而喘       │
│  C. 风热表证   - 咽痛口渴       │
│  D. 暑湿表证   - 身重困倦       │
└─────────────────────────────────┘

[选择A/B/C/D] 或直接输入"风寒表实"
```

#### 交互形式2：开放式问答（适合进阶）

```
青木先生：「徒儿，麻黄与桂枝均能解表，二者有何不同？」

┌─────────────────────────────────┐
│  [请输入你的理解...]             │
│                                 │
│  提示：可从性味、功效、适应症   │
│        等角度对比               │
└─────────────────────────────────┘

玩家输入：「麻黄辛温，发汗力强，主治风寒表实证；
          桂枝辛甘温，发汗力缓，主治风寒表虚证」
```

#### 交互形式3：情境对话（最自然）

```
青木先生：「为师考考你。若见恶寒发热，头痛身痛，
         无汗而喘，舌苔薄白，脉浮紧，当用何方？」

玩家输入：「麻黄汤」

青木先生：「善！麻黄汤发汗解表，宣肺平喘，正对此证。
         方中麻黄为君，桂枝为臣，你可知道为何？」

玩家输入：「不知道」

青木先生：「无妨。麻黄辛温发汗，为解表之要药；
         桂枝通阳助麻黄发汗，为臣辅之。
         二者相须为用，解表之力倍增。
         记住了吗？」
```

#### 交互形式4：苏格拉底式引导

当玩家答错时，不直接给出答案，而是引导思考：

```
青木先生：「再想想。无汗而喘，脉浮紧，提示邪气在表还是入里？」

玩家输入：「在表」

青木先生：「对。邪气在表，当用汗法。但病人无汗，
         说明腠理闭塞，需要用发汗力强的药还是弱的呢？」

玩家输入：「强的」

青木先生：「正是。麻黄发汗力强，桂枝发汗力缓。
         无汗而喘之表实证，当用麻黄为君。
         懂了吗？」
```

### 4.3 智能出题系统

**出题逻辑**：

```typescript
interface QuestionGenerator {
  generateQuestion(params: {
    chapterId: string;           // 当前章节
    currentStage: number;        // 当前阶段（识药/方剂/临床）
    collectedMedicines: string[]; // 本章已收集药物
    askedQuestions: string[];     // 已问过的问题ID（防重复）
    wrongHistory: string[];       // 答错题ID（重点复习）
    questionType: 'single' | 'compare' | 'formula' | 'clinical';
  }): Question;
}
```

**题目类型分布**：

| 类型 | 占比 | 说明 | 示例 |
|------|------|------|------|
| **单药识别** | 30% | 根据线索猜药 | "辛温，归肺经，能发汗解表的是？」 |
| **药对对比** | 25% | 两药对比差异 | "麻黄与桂枝的区别？」 |
| **方剂应用** | 25% | 方剂组成/主治 | "麻黄汤的组成是？」 |
| **临床辨证** | 20% | 病案分析 | "恶寒发热，无汗而喘，当用何方？」 |

**防重复机制**：
- 每题有唯一ID，记录到`askedQuestions`
- 同章节内不重复出相同题目
- 优先出未覆盖的知识点
- 错题3题后再次出现（间隔复习）

**进度追踪**：

```typescript
interface LearningProgress {
  chapterId: string;
  medicineCoverage: Record<string, number>; // 每味药出题覆盖度
  topicCoverage: Record<string, number>;    // 各知识点覆盖度
  totalQuestions: number;      // 本章总题数
  correctRate: number;         // 正确率
  weakPoints: string[];        // 薄弱知识点
  masteryLevel: number;        // 掌握度 0-100
}
```

## 五、核心玩法保留与增强

### 5.1 性味归经探查（v2.0核心）

**水晶球系统保留**：
- 真实药图展示
- 五层探查（药图→四气→五味→归经→功效）
- 消耗方灵石获取线索
- 根据线索猜测药名

**v3.0增强**：
- AI导师根据探查进度给出提示
- 探查策略推荐（"可先探查四气，缩小范围"）
- 收集后导师讲解药材故事

### 5.2 亲密度系统（v2.0保留）

| 等级 | 分数 | 称号 | 解锁内容 |
|------|------|------|----------|
| Lv.1 | 0-20 | 初识 | 基础信息 |
| Lv.2 | 21-40 | 相识 | 功效详情 |
| Lv.3 | 41-60 | 熟悉 | 药材故事 |
| Lv.4 | 61-80 | 亲密 | 配伍记忆 |
| Lv.5 | 81-100 | 知己 | 专属方歌 |

### 5.3 方剂追缉令（改编为章节任务）

- 每章的方剂学习即为"追缉任务"
- 收集齐组成药物后"解锁"方剂
- 不再作为独立每日任务

### 5.4 临床实习（保留为核心考核）

- 每章最后的Boss考核
- 必须正确辨证+选方+选君药
- AI导师现场点评

